# Libasm

This project implements some of the functions of the C standard library in Assembler.


## Function list:
- `atoi_base`
- `list_front`
- `list_sort`
- `list_size`
- `read`
- `remove_if`
- `strcpy`
- `strcmp`
- `strdup`
- `write`

## How work

To use the functions, compile the project with Makefile and connect header libasam.h in you project.
